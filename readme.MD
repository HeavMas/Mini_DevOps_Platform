# 🚀 Mini_DevOps_Platform

Локальная DevOps-платформа, имитирующая production-инфраструктуру с CI/CD, мониторингом, логированием и проксированием. Проект разработан с учётом ограниченных ресурсов и полностью разворачивается на одной машине.

---

## 📌 Описание проекта

Этот проект представляет собой self-hosted инфраструктуру, включающую:

* деплой приложений через контейнеры
* reverse proxy
* централизованный сбор метрик
* централизованное логирование
* CI/CD пайплайн

Цель — воспроизвести реальные DevOps-практики в локальной среде.

## 🚀 Быстрый старт

```bash
git clone <repo>
cd project

cp .env.example .env

docker-compose up -d
```

После запуска:

* Grafana → [http://localhost:3000](http://localhost:3000)
* Prometheus → [http://localhost:9090](http://localhost:9090)
* App → [http://localhost/api](http://localhost/api)
